# Youtube video streamer

## Installation

```
$ npm install
```

## Build

```
$ npm run build
```

## Start

```
$ npm run start
```

## Test

```
$ npm test
```

## Docker image

Build:
```
$ docker build . -t youtube-audio-streamer
```

Run container:
```
$ docker run --rm -it -p 3000:3000 youtube-audio-streamer
```
